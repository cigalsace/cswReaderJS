# Version 0.06

13/01/15:
    - La recherche ne tient plus compte de la casse (majuscules/minuscules)
    - L'opérateur "+" peut être utilisé comme notation du "et", ainsi:
        - "Première guerre mondiale" = rechercher les fiches contenant le mot "Première guerre mondiale"
        - "munster" = rechercher les fiches contenant le mot "munster"
        - "munster + 1916" = rechercher les fiches contenant le mot « munster » et « 1916 »

A noter, la recherche se fait en texte brut sur la fiche sans notion de pertinence. Ainsi, une recherche sur "20000" sortira toutes les cartes car "20000" est contenu dans le code SIREN du PAIR qui est inclu  dans l'identifiant de chaque fiche...
